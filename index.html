peer2penguin

peer2penguin is the name of a PeerJS wrapper that allows you to use three variables in the global namespace: incomingDict, outgoingDict, and iAmHost.

iAmHost is a boolean that is updated to false if you connect to a remote Peer's ID.

incomingDict is a JS object that holds the data that your peer has sent you. It updates every 10ms.

outgoingDict is a JS object that holds the data that you are sending your peer. It updates every 10ms.
